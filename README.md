# Phaser Typescript Starter 


## Clone project
```
    git clone https://github.com/jimmycarry/typescript-phaser-starter.git
```

## Install

```
npm install
```


## start dev
```
 npm start
```

## Open In Browser
open browser in localhost:1234

## deployment
```
npm run build
```
and then push prod directory to your web server directory
